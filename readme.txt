== Description ==
TiengViet.IO là hệ thống giúp người dùng tối ưu nội dung tiếng Việt để làm SEO Google. Với plugin này bạn có thể tìm kiếm từ khóa ngữ nghĩa liên quan (LSI Keyword), kiểm tra bài viết chuẩn SEO và nhiều tùy chọn khác nữa.

TiengViet.IO là một nhánh ứng dụng của dự án xử lý ngôn ngữ tự nhiên được áp dụng trong ngành SEO - Marketing. Đây là nền tảng hỗ trợ người làm SEO hoàn toàn do người Việt xây dựng. Chúng tôi tập trung chủ yếu vào phân tích nội dung.

Tại plugin TVAudit, chúng tôi có sử dụng API để truyền và lấy thông tin từ website https://api.tiengviet.io/my/ để giao tiếp dữ liệu. Các trường hợp sử dụng API

- Lấy thông tin tài khoản
- Nạp tiền vào tài khoản
- Thêm mã giảm giá
- Lấy thông tin từ khóa LSI
- Lấy thông tin kết quả tối ưu bài viết

Xem thêm về điều khoản sử dụng của TiengViet.IO tại: https://my.tiengviet.io/policies.html

== Installation ==

Nếu bạn cài từ WordPress Plugins Store

1. Truy cập **Plugins > Add New**
1. Tìm kiếm **TiengVietIO**
1. Click vào nút **Install Now** và cài đặt
1. Click vào nút **Activate** để kích hoạt plugin

Nếu bạn cài từ file download
1. Tải file TVAdit.zip và giải nén
1. Tải các file đã giải nến lên thư mục '/wp-content/plugins/'
1. Vào menu 'Plugins' trong WordPress và tiến hành kích hoạt Plugin TiengVietIO
